# MFARM project 
Some place without stakes like Linkedin or Medium. Some place bordering between a personal journal and a blog.

## Joining the web revival initiative
Places like Neocities wnat you to enjoy internet again. Once upon a time we imagined a reality where everyone will have their own website and we would visit them like real places, getting to know people, building virtual 'neighbourhoods'. If this is something that sounds interesting to you visit places that inspired me to start this journey:
- [A Web Revival: the Internet didn't die, you're just not on it ](https://www.youtube.com/watch?v=tkUgOT22F5s)
- [Intro to the Web Revival](https://thoughts.melonking.net/guides/introduction-to-the-web-revival-1-what-is-the-web-revival)

## Commands

All commands are run from the root of the project, from a terminal:

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | Installs dependencies                            |
| `npm run dev`             | Starts local dev server at `localhost:4321`      |
| `npm run build`           | Build your production site to `./dist/`          |
| `npm run preview`         | Preview your build locally, before deploying     |
| `npm run astro ...`       | Run CLI commands like `astro add`, `astro check` |
| `npm run astro -- --help` | Get help using the Astro CLI                     |
